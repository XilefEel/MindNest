# MindNest

MindNest is a desktop productivity app built with Tauri and React inspired by Obsidian, Trello, and Notion.

## Tech Stack

### Frontend

- React
- TypeScript
- Tailwind CSS
- ShadCN/UI
- React Router
- Zustand
- dnd-kit

### Backend

- Tauri (Rust)
- SQLite

## Getting Started

To run the app locally:

```bash
npm install
npm run tauri dev
```
